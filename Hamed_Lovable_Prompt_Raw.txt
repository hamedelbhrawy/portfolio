===================================
DESIGN SYSTEM & VISUAL DIRECTION
===================================

Design Philosophy: "Cyber Corporate" — the clean sophistication of premium consulting brands combined with the technical edge of data infrastructure tools. Dark hero sections establish authority. Light content sections ensure readability. Gold accents signal premium positioning.

Color Palette:
- Primary Background (Hero/Nav/Footer): #0F172A (Midnight Navy)
- Accent / CTA Buttons: #C5A55A (Rich Gold)
- Secondary Accent (Tags/Badges/Code): #1ABC9C (Electric Teal)
- Tertiary Accent (Links/Charts): #3B82F6 (Electric Blue)
- Content Background: #FFFFFF (White)
- Alternating Sections: #F1F5F9 (Soft Slate)
- Body Text: #333333 (Charcoal)
- Muted Text: #94A3B8 (Light Gray)

Typography:
- Headings (H1-H3): Plus Jakarta Sans — Bold/SemiBold/Medium
- Body Text: DM Sans — Regular (400)
- Metric Numbers & Code: JetBrains Mono — Bold for metrics, Regular for code
- Arabic Text (for bilingual toggle): IBM Plex Sans Arabic
- H1 Desktop: 48-56px | Mobile: 32-36px
- H2 Desktop: 32-36px | Mobile: 24-28px
- Body Desktop: 16-18px | Mobile: 15-16px
- Metric Numbers Desktop: 48-72px | Mobile: 36-48px

Layout Rules:
- Max content width: 1200px, centered
- Section padding: 80-120px vertical desktop, 40-60px mobile
- Card grids: 3 columns desktop, 2 tablet, 1 mobile, 24px gap
- Hero sections: 100vh on homepage, 60vh on inner pages
- Asymmetric layouts: alternate text-left/visual-right with text-right/visual-left
- Wide layout with generous whitespace — avoid cramped density

Navigation:
- Sticky top navigation bar with glass-morphism effect (frosted glass on dark navy)
- Primary items: Home, About, Services, Portfolio, Blog, Contact
- "More" dropdown: Case Studies, Resources, Media, Testimonials
- Persistent gold CTA button top-right: "Book a Call"
- EN/AR language toggle in top-right corner
- Collapses to hamburger menu on mobile

Animations & Motion:
- Scroll-triggered fade-up animations (staggered 100ms between elements) using Framer Motion
- Animated metric counters that count up from 0 on scroll into view
- Horizontally scrolling tech stack logo marquee (continuous loop)
- Card hover: translateY(-4px) with increased box-shadow
- Typed text animation on hero subheadline cycling through: "Data Platform Architect" | "BI Strategy Leader" | "Analytics Engineer"
- Career timeline nodes appear sequentially as user scrolls
- Page transitions: gentle 300ms fade
- All motion should feel premium and intentional, not playful

Visual Assets:
- Use Lucide icons exclusively throughout the site
- No stock photos anywhere — use custom geometric shapes, gradient meshes, and abstract data-flow patterns for backgrounds
- Dashboard mockup placeholders for portfolio items
- Architecture diagram placeholder for the About page
